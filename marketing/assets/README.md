# Marketing assets — drop logo files here

Required:
- `smartsoft-logo.png` — transparent PNG, used on landing + pricelist + footer.

Recommended variants (used if present):
- `smartsoft-logo.svg` — vector for crisp scaling.
- `smartsoft-logo-onblack.png` — white-on-black variant for dark surfaces.
- `favicon.ico` — for the landing page tab icon.

If `smartsoft-logo.png` is missing, the landing and pricelist fall back to a styled "Smart**soft**" wordmark (black text + yellow accent) so they still render cleanly.
